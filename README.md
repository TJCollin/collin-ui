# CollinUI
CollinUI, a React based component library for developers, designers and product managers.

## Install
```javascript
npm install collin-ui --save

// or

yarn add collin-ui
```

## Quick Start
``` javascript
// import css globally
import 'vikingship/dist/index.css'

import {
  Input,
  Button
  // ...
} from 'collin-ui'

```
